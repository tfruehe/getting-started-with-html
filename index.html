Session 0 Basics of the Web

World Wide Web is a Collection of 30 Billion Linked HTML Pages.

Browser retrieves HTML files via HTTP from Servers.

HTML Hypertext Markup Language

Text is what you see

Markup is a Tag: An HTML tag is always contained within angled brackets.
Most tags have an opening tag and a closing tag.
Some tags (called "void" tags) do not require a closing tag.
Inline Tags: adds line example:
a, span, br , img, strong
Block Tags: creates box has height and width example:
div, form, p

Element: An HTML element refers to everything within a set of opening and closing tags

Attribute: This is a property of an HTML element.
